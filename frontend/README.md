# Frontend

## Local Setup

1. Install dependencies:

```bash
bun install
```

2. Start dev server:

```bash
bun run dev
```

By default, local development uses:

`VITE_BACKEND_URL=http://127.0.0.1:8787`

from `.env`.

## Environment Files

- `.env`: local backend URL

## Build

```bash
bun run build
```
